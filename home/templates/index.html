Description of the Web App

This web application was made to test various abilities using Python, like webscraping data, making maps, creating graphs and making a web application to present all this.

The Map Updates itself every hour in a development server (https://github.com/basiliofarach/covid_webscraper_map), on production (Heroku - https://covid-webscraper.herokuapp.com/) to perform this is needed to making the data classes and to make a type of store that updates, but it overpasses the meaning of this Web API.

At the start the features of the map are deactivated, it shows only a plain map, you have to activate the features.

The current data used is presented in the header, and it's obtained from "www.worldometers.info/coronavirus/", all credits to that page, the data can be obtained from a different API, but I wanted to test how to make a webscraper and adjust the data to my needs.
